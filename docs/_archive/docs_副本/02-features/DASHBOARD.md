**设计规范**:

**内容显示**:
- [ ] 标题："🤖 Now Engine 推荐你现在做："

- [ ] "第一步：xxx"（蓝色小字）
- [ ] 情绪标签 😊😐😰

**按钮**:
- [ ] 主按钮：[🚀 开始专注] — 蓝色背景，白色文字，印刷风阴影

- [ ] 次按钮：[换一个] — 灰色边框
  - 点击行为：Now Engine 按优先级顺序推荐下一个任务

**空状态**:
- [ ] "今天没有任务，去添加一个吧 →" + [+ 添加任务] 按钮

### ✅ Guardian 自动弹窗

#### Morning Ritual 模态框

**触发条件**: 每日 5:00-18:00 首次打开应用，且开关已开启

**设计**: 风格 A1 紫色边框，全屏居中模态框

**内容**:
- 标题："☀️ 早上好！让我们规划今天"
- 子标题：日期 + 星期
- 今日已有时间块预览
- 未完成任务预览（最多 3 个）
- 输入框："今天第一个要完成的是：___"（预填 Now Engine 推荐）
- 按钮：[✓ 开始今天]

**关闭行为**: 记录 `lastMorningRitualDate = 今天`，当天不再显示

---

#### Daily Review 模态框

**触发条件**: 每日 20:00 后首次打开应用，且开关已开启

**设计**: 风格 A1 绿色边框，全屏居中模态框

**内容**:
- 标题："🌙 今天辛苦了！来复盘一下"
- "🎉 今天完成了：X 个任务，专注了 Y 小时"
- "计划 vs 实际：完成度 Z%"
- 输入框："明天最重要的一个任务是：___"
- 按钮：[✓ 明天加油]
