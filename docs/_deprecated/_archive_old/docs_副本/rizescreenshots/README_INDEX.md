### ⚙️ 设置页面信息架构

Rize 设置页面采用左侧导航分组，共 5 大模块：

1. **ACCOUNT & BILLING** - Account, Billing, Org Members
2. **WORKSPACES** - 多工作区切换（个人/团队）
3. **TRACKING** - Activity, Breaks, Categories, Meetings, Time Entries, Tracking Rules
4. **PRODUCTIVITY** - Coach, Distraction Blocker, Focus, Planning
5. **INTEGRATIONS & DATA** - API, Calendars, Data Export, Integrations, Reports
6. **PREFERENCES** - Notifications, Privacy, Theme

### ⚠️ Trace MVP 不做的功能

- 团队协作 / 多工作区 (Rize Pro 功能)
- 积分 / 排行榜游戏化
- 好友对比
- 复杂的 PDF 报告导出
- Google OAuth 登录（MVP 使用手机号/微信）
- Client/Project 层级（MVP 只有 Tasks）
- Urge Surfing 成瘾拦截模式
