## 📝 技术说明

### 为什么用原生 SVG 而不是图表库？
1. **性能** — 简单饼图和柱状图用原生 SVG 只有几十行代码，比引入整个 Chart.js 轻量得多
2. **控制** — 可以精确匹配双层马卡龙设计系统的配色和圆角
3. **无依赖** — 不增加包体积，不引入第三方依赖的安全风险
