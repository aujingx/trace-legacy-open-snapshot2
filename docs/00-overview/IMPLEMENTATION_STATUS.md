### ⚠️ 部分工作，但需要立刻修的问题

#### 问题 1：数据双轨制
- **现状**：自动追踪写 `activities_YYYY-MM-DD.json`，手动时间块写 SQLite `time_blocks` 表
- **用户痛点**：统计页和时间线看到的不是同一套数据
- **修复优先级**：🔴 P0

#### 问题 2：打包后数据目录漂移
- **现状**：活动 JSON 用 `dirs::data_dir()`，SQLite 用 `app_config_dir()`，dev 和 prod 路径可能不一致
- **用户痛点**：开发时录的数据，打包后看不到；或者反过来
- **修复优先级**：🔴 P0

#### 问题 3：首次启动后 is_tracking 可能是 false
- **现状**：App 初始化时可能没正确启动追踪
- **用户痛点**：用了一小时才发现根本没在录
- **修复优先级**：🔴 P0
